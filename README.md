# Programming & Web Development @ The English High School

A website for the Programming & Web Development Chapter 74 Career Technical Education program at The English High School.

## Live Site

[https://ehspwd.github.io/pwd-website/](https://ehspwd.github.io/pwd-website/)

## Pages

- **Homepage** (`index.html`) — Program overview, areas of study, four-year course pathway (Grades 9–12), industry certifications, student projects, learning environment, and career pathways after graduation.
- **Showcase** (`showcase.html`) — Freshmen Exploratory Student Portfolio Showcase featuring student websites built during a 4-week introduction to HTML and CSS. Students are organized by cohort with links to their individual projects.

## Tech Stack

- HTML
- CSS
- JavaScript

## Project Structure

```
website/
├── index.html            # Homepage
├── showcase.html         # Freshmen Exploratory Showcase
├── styles.css            # Shared stylesheet
├── script.js             # Shared JavaScript (scroll animations, nav toggle)
├── assets/
│   └── images/           # Hero background photos
└── cohort/
    ├── 01/               # Cohort 1 student projects
    ├── 03/               # Cohort 3 student projects
    ├── 04/               # Cohort 4 student projects
    └── 05/               # Cohort 5 student projects
```

## Features

- Responsive design for desktop and mobile
- Scroll reveal animations
- Interactive timeline for the four-year course pathway
- Student project showcase with linked card grid per cohort
- Blue gradient hero sections with background imagery
