Course Sequence

Your Four-Year Pathway

A structured journey from exploration to specialization.

Grade 9 — Exploratory

Introduction to Computing & Web Development

Students are introduced to computing and web development fundamentals.

  • Introduction to HTML & CSS
  • Web page structure and styling
  • Digital literacy and safe computing practices
  • Creative problem solving through introductory web projects
Grade 9 — Year 1

Foundational Technical Skills

Students develop foundational technical skills used across all technology careers.

  • History of computers and the software industry
  • Computer hardware and system components
  • Building and maintaining computers
  • Command Line Interface (CLI) navigation
  • Git & GitHub for version control
  • Workplace ergonomics and lab safety
Grade 10

Web Development Foundations

Students focus on building structured, accessible, and responsive websites.

  • Semantic HTML and accessibility best practices
  • CSS layout, styling, and responsive design
  • UI design fundamentals and usability principles
  • Multi-page website development
  • Web standards, validation, and optimization
Grade 11

JavaScript & Interactive Development

Students bring logic and interactivity to their applications.

  • JavaScript fundamentals and programming logic
  • Variables, conditionals, loops, functions, arrays, and objects
  • DOM manipulation and event handling
  • Debugging and error handling
  • Client-side form validation
  • Interactive web applications and team-based projects
Grade 12

Advanced Topics & Specialization

Students apply their skills to advanced, career-focused areas.

  • Cybersecurity fundamentals and secure systems
  • Game development concepts and logic
  • Agile project management and team workflows
  • Software testing and deployment concepts
  • Capstone-style projects and portfolio development
